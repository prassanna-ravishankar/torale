# Torale Brand Assets

## Directory Structure

```
brand/
├── svg/
│   └── logo.svg          # Vector logo (scalable, use for web)
└── png/
    ├── logo-32.png       # 32x32px (favicon, small icons)
    ├── logo-64.png       # 64x64px (icons)
    ├── logo-128.png      # 128x128px (thumbnails)
    ├── logo-256.png      # 256x256px (medium icons)
    ├── logo-512.png      # 512x512px (large icons, PWA)
    └── logo-1024.png     # 1024x1024px (high-res, print)
```

## Usage Guidelines

### When to Use SVG
- Web applications (preferred for scalability)
- Any digital interface where vector support is available
- Responsive designs that need to scale

### When to Use PNG
- Favicons and app icons
- Email signatures
- Social media profiles
- Documentation where SVG isn't supported
- Any raster-based medium

### Logo Typography
- Logo text: **τorale** (tau + orale, all bold)
- τ = lowercase Greek tau
- orale = lowercase, bold weight

### Color Palette
Primary colors from logo:
- Dark: #203345 (primary logo color)
- Light Gray: #CDD1CF
- Medium Gray: #D8DCD9, #D6D9D5, #C8CCCB
- Accent: #3E4D59

### Clear Space
Maintain minimum clear space of 20px around logo on all sides

### Do Not
- Distort or skew the logo
- Change logo colors (use provided variations)
- Place on busy backgrounds without proper contrast
- Scale below 32px height
