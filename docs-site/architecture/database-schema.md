# Database Schema

Full schema documented in [CLAUDE.md](https://github.com/torale-ai/torale/blob/main/CLAUDE.md#database-schema).

## Key Tables

- `users` - Clerk-authenticated users
- `api_keys` - CLI authentication keys
- `tasks` - Monitoring tasks with schedules
- `task_executions` - Execution history
- `task_templates` - Pre-built templates
