# User Guide

Learn how to use Torale's web dashboard to create and manage monitoring tasks.

## Quick Navigation

**[Dashboard Overview](./dashboard)** - Main interface sections, status indicators, and quick actions

**[Creating Tasks](./creating-tasks)** - Task creation wizard, templates, and configuration

**[Managing Tasks](./managing-tasks)** - Edit, pause, resume, and delete tasks

## Getting Started

New to Torale? Start with the [Dashboard Overview](./dashboard) to understand the interface, then follow the [Creating Tasks](./creating-tasks) guide to create your first monitoring task.
