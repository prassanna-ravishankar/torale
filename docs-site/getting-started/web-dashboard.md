# Web Dashboard

Use Torale through the web interface at [torale.ai](https://torale.ai).

## Sign Up

1. Visit [torale.ai](https://torale.ai)
2. Sign up with Google, GitHub, or email
3. Verify your email (if using email signup)

## Create Your First Task

The dashboard provides a multi-step wizard for creating monitoring tasks.

### Using Templates

1. Click "New Task"
2. Select a template category (Product Releases, Price Tracking, etc.)
3. Fill in placeholders (product name, price threshold, etc.)
4. Set schedule
5. Create task

3. Describe your trigger condition
4. Configure schedule using the visual builder
5. Preview results (optional)
6. Create task

## Managing Tasks

**View tasks** - See all active and paused tasks in the dashboard

**Edit tasks** - Update query, condition, schedule, or notification settings

**Pause/Resume** - Temporarily disable tasks without deleting them

**Delete tasks** - Remove tasks you no longer need

## Viewing Results

**Execution History** - See every time a task ran, with status and duration

**Notifications** - View only executions where your condition was met

**Sources** - Click through to the web pages where information was found

## Notification Settings

Configure how you want to be notified:

**once** - Get notified when condition first becomes true, then pause the task

**always** - Get notified every time the condition is met
